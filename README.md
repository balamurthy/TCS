
Playwright hands on
1. Install playwright
Practice the .ts files by downloading into Typscript folder

2.Open the vscode from Playwright folder
Open the first test 
Run the test both ways using tests in the left navigation 
And issuing the npx playwright test tests/sample.spec.ts

3. Use npx playwright codegen to record a login test on saucedemo.com
4. Use grep to run tests tagged with @Positive, @Negative etc using grep command
Example 
 npx playwright test/login.spec.ts --grep "@Positive"
