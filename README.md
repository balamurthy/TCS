Hands on practice assignment in Cucumber with TestNG

1.Create a maven project , add the pom.xml i provided in this branch.
2.Add the TestRunner in and TestNG.xml to the project folder
3.create feature files login.feature and search.feature file. Write gherkin syntax steps in them
4.Run each feature file and implement the missing steps in respective java class files
5.Implement page object model by creating pages package and placing the Pages and call within the stubs . 
6.Assert inside the step definitions in the last validation step of the feature 
7. Run the testNG.xml , which will run the Test runner and note the results.
8. Result html file ,json file should be available in target folder in the path of the plugin given in testrunner
